Gritano-OpenSSH
===============

This is a patched OpenSSH version used by Gritano that enables SSH lookup for public keys in a database.

- [OpenSSH](http://www.openssh.com/)
- [OpenSSH for Git](https://github.com/wuputahllc/openssh-for-git)
